PROJECTS

clyt [v1.0.0]

clyt v 1.0.0 is a minimalistic YouTube downloader tool. It allows you to enter a YouTube URL, select the output format (audio/video), and choose the output directory.

Technologies used: Python, Tkinter

CLyT [v2.0.0]

CLyT v2 builds on the features of v1, adding functionality tailored for DJs and power users. In addition to existing features, it supports batch downloads of audio/video, generates single and batch download lists (.txt), and includes a space for writing song requests. This enhanced version meets the complex requirements of professional use.

Technologies used: Python, Tkinter
