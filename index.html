Our Projects

CLyT v1

CLyT v1 is our initial version of the YouTube downloader. It allows users to download videos from YouTube in various formats. Built using Python and Tkinter, it provides a simple and user-friendly interface for downloading videos easily.

Technologies used: Python, Tkinter

CLyT v2

CLyT v2 is an enhanced version of our YouTube downloader, offering additional features such as batch downloads, format selection (MP3 or MP4), and improved logging of download statuses. This version continues to use Python and Tkinter for a robust and efficient application.

Technologies used: Python, Tkinter
